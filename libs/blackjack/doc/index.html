Monte Jack


A Java API for playing Blackjack


This set of classes provides free and open access to:

  • Classes representing playing cards, including basic abstractions for non-standard decks
  • Detailed logic to play Blackjack
  • A simple reference implementation that allows you to exercise these classes via the console window

Unlike numerous Blackjack APIs, this implementation provides for playing using virtually all variations of basic Blackjack, including changing:

  • If the dealer stands or hits on soft 17s
  • The number of decks in the shoe
  • Whether doubling down after split is allowed
  • Whether doubling down is restricted to hands of 9s, 10s, and 11s
  • The Blackjack payoff multiplier
  • If early surrender is allowed
  • If late surrender is allowed
  • Whether double after split is allowed
  • If resplitting aces is allowed
  • The maximum number of splits
... as well as a few other options.

I originally wrote this library to study the mathematics of Blackjack by performing Monte Carlo simulations on various playing strategies.

Downloading

You can download Monte Jack version 1.0.1, including source, in tarball and ZIP formats from garrettsmith.net.

Using Monte Jack

To implement a Blackjack player, first implement the EventHandler interface, then: 

double wager = 1000; // you're a high roller, right?
EventHandler eventHandler = new MyEventHandler();
Blackjack blackjack = new Blackjack();
blackjack.playGame( eventHandler, wager );

You can also play Blackjack via the console (using EventHandler) by using the scripts in the bin directory.
