Provides classes for playing a game of blackjack.

Implement {@link garrettsmith.blackjack.EventHandler} to respond to events that a fired by {@link garrettsmith.blackjack.Blackjack}. Then, pass in an instance of that handler to {@link garrettsmith.blackjack.Blackjack#playGame}.

For example: 

        double wager = 2.0;
        Blackjack blackjack = new Blackjack();
        EventHandler handler = new MyEventHandler();
        blackjack.playGame( handler, wager );

See {@link garrettsmith.blackjack.refimpl.TextPlayer} for an example implementation.
